# E-commerce VStore - Backend Architecture

Sistema backend desenvolvido para gestão de lojas virtuais.

## Arquitetura
- Spring Boot
- PostgreSQL
- Docker
- Deploy no Render
- Autenticação JWT
- Controle de acesso por perfil
- Arquitetura em camadas

## Responsabilidades técnicas
- Modelagem de banco
- Implementação da API REST
- Segurança com JWT
- Deploy automatizado
- Tratamento global de exceções
- Padronização de respostas
